C# - Biography

C# is a programming language developed by Microsoft and released in 2000 as part of the .NET framework. It is designed to be a modern, simple, and object-oriented language with a syntax similar to C++ and Java. C# is widely used for desktop application development, web services, mobile applications with Xamarin, and more.

C# Keywords

using: This keyword is used to import namespaces in C#. It allows the programmer to access classes and methods defined in other libraries. For example, using System; allows the use of the Console class to print text to the console.

namespace: It is a way to organize code in C#. Namespaces allow grouping related classes and avoiding name conflicts. For example, namespace MyApplication is useful for encapsulating classes of a specific application.

class: Defines a new class in C#, which is the fundamental structure of object-oriented programming. For example, public class Person defines a Person class that can have properties and methods.

public: It is an access modifier that indicates that a member (class, method, or variable) is accessible from anywhere in the code. It is commonly used when declaring classes and methods, for example, public void ShowMessage().

void: It is used in method declarations to indicate that the method does not return any value. For example, public void Greet() defines a method that performs an action but does not return data.
